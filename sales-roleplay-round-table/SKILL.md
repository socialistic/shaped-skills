# 保险销售话术陪练 — 小圆桌

基于 [CheungZeeCn/LittleRoundTable](https://github.com/CheungZeeCn/LittleRoundTable)
的对话管理架构，将保险销售话术培训转化为人机对话陪练。

> 原始仓库提供 Python 实现的多分支对话树引擎、逐句关键词评分和异议注入机制。
> 本技能将该架构抽象为一个可直接使用的 LLM 陪练工作流，无需部署 BERT 服务或 Python 后端。

## 用户任务

帮助保险销售新人（初级代理人）通过模拟客户对话练习约访话术，获得逐句反馈和评分。

## 输入

用户提供以下任一组合：

1. **话术脚本**（JSON 或自然语言描述）— 包含对话分支、标准话术、相似话术变体、
   奖励/扣分关键词。若未提供，使用内置的约访话术模板。
2. **练习模式** — 学习模式（逐句提示标准答案）、练习模式（仅在偏离时提示）、
   考核模式（全程不提示，结束后统一评分）。默认：练习模式。
3. **自定义异议**（可选）— 额外的客户异议场景，注入到对话流程中。

## 对话树架构

话术脚本遵循四层结构：

```
Script → Segment → TurnBlock → Answer
```

- **Script**: 一套完整话术（如"电话约访"），包含元信息和策略指导
- **Segment**: 对话阶段（如"开场"、"约访"），有单一入口和出口
- **TurnBlock**: 对话节点，通过 parent_ids/children_ids 形成有向图
  - 类型: start / default / objection / end
  - 每个节点包含: standard_answer, similar_answer[], bonus_keywords[], minus_keywords[], tips
- **Answer**: 标准回答及其变体，用于意图匹配和评分

## 开场策略（内置模板）

1. **请求帮忙法** — "我们公司正在做市场调研，能否请您帮个忙？"
2. **第三者介绍法** — "是您的朋友X先生介绍我打电话给您的…"
3. **牛群效应法** — "上个月，您的同事赵X在我们公司购买了一款保险产品…"
4. **巧借东风法** — "您上个月给我们公司打过咨询电话…"

## 工作流

### Step 1: 加载话术

读取用户上传的话术脚本，或使用内置的约访话术模板。解析对话树结构，
识别所有 TurnBlock 及其父子关系，构建对话状态图。

### Step 2: 确定练习模式

根据用户选择设置陪练行为：

- **学习模式**: 每轮显示当前节点的 standard_answer 和 tips，用户跟读后评分
- **练习模式**: 仅在用户回答偏离所有 similar_answer 变体时提示，鼓励自主发挥
- **考核模式**: 全程不提示，记录每轮得分，结束后输出完整评分报告

### Step 3: 开始对话陪练

1. 从 start 类型节点开始，扮演客户角色说出 default_replies
2. 等待用户（销售代理人）回答
3. 将用户回答与当前节点的 standard_answer 和 similar_answer 进行语义匹配
4. 检查 bonus_keywords（加分）和 minus_keywords（扣分）
5. 根据匹配结果评分，给出即时反馈（学习/练习模式）
6. 沿对话树推进到下一个 TurnBlock

### Step 4: 异议注入

在对话过程中，当遇到 objection 类型节点时：

- 模拟客户突然提出异议（如"我没有时间"、"我已经有保险了"）
- 评估用户的异议处理话术
- 根据处理质量决定对话是否继续推进或需要重试

### Step 5: 评分与总结

对话结束后输出：

- **总分**: 各节点得分加权汇总
- **逐句评分明细**: 每个 TurnBlock 的得分、匹配的关键词、偏离点
- **改进建议**: 基于 minus_keywords 命中和未覆盖的 bonus_keywords 给出具体改进方向
- **对话路径回放**: 展示实际走过的分支路径，对比最优路径

## 输出合约

最终输出为结构化的陪练报告，包含：

1. 练习模式和话术脚本名称
2. 总分及等级（优秀/良好/及格/需改进）
3. 逐句评分表
4. Top 3 改进建议
5. 对话完整记录（带标注）

## 关键约束

- 陪练角色始终扮演客户，不代替用户回答销售话术
- 评分基于关键词匹配和语义相似度，不做主观判断
- 异议注入频率和难度与练习模式匹配（学习模式少注入，考核模式按脚本完整注入）
- 所有反馈使用中文，匹配目标用户群体（中国保险行业初级代理人）
