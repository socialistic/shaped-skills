# 한국사 History Teller RAG 챗봇

한국사 수능 및 한국사능력검정시험 대비를 위한 RAG 기반 질의응답 스킬입니다.
사용자가 한국사 문제(주관식·객관식)를 올리면, 교과서·사료 청크에서 BM25 검색 후
재순위(Jaccard) 상위 문맥을 GPT에 전달해 정확한 풀이와 역사적 배경을 제공합니다.

## 출처 (Provenance)

- **원본 저장소**: <https://github.com/minsung6333/history_teller_rag_chat>
- **소개 글**: <https://velog.io/@minsung6333/Project-RAG%EB%A5%BC-%ED%99%9C%EC%9A%A9%ED%95%9C-%ED%95%9C%EA%B5%AD%EC%82%AC-chatbot-%EB%A7%8C%EB%93%A4%EA%B8%B0-0>

원본 저장소에는 유효한 `SKILL.md`가 없으며, 이 스킬은 공개 워크플로를
socialistic.ai 호환 스킬로 변환한 것입니다.

## 데이터 소스

원본 프로젝트가 사용하는 코퍼스:

- 제6차·제7차 교육과정 한국사 교과서
- 2022–2024학년도 수능 한국사 기출문제
- 관련 사료·역사 기록

## 워크플로

1. **문제 업로드**: 사용자가 한국사 기출문제 이미지, 텍스트, 또는 직접 질문을
   입력합니다.
2. **검색 (Retrieval)**: BM25 (Okapi) 알고리즘으로 교과서·사료 청크 25개를
   1차 검색합니다.
3. **재순위 (Re-ranking)**: Jaccard 유사도로 상위 5개 문맥을 선별합니다.
4. **풀이 생성**: 선별된 문맥과 함께 GPT에 전달하여 답변·오답 해설·역사적
   배경을 생성합니다.
5. **후속 질문**: 같은 세션에서 추가 질문이나 심화 설명을 요청할 수 있습니다.

## 입력

- **파일**: 한국사 기출문제 이미지 또는 텍스트 파일 (수능, 모의고사, 한능검)
- **텍스트**: 한국사 관련 질문 — 주관식("임진왜란의 원인과 전개를 설명해") 또는
  객관식("다음 중 통일신라의 특징은?")

## 출력

- 정답과 근거(교과서·사료 출처 포함)
- 객관식인 경우 각 선지별 옳고 그른 이유
- 관련 역사적 배경·맥락 설명
- 오류가 있는 질문에 대한 교정

## 제약사항

- 한국사 범위 외 질문에는 답변하지 않습니다.
- 풀이는 참고자료(교과서·사료)에 근거하며, 자료가 부족한 경우 모델 지식으로
  보충하되 명시합니다.
